# gameboy1989
Gameboy1989 is an Gameboy Emulator to emulate a 1989 and pre-1989 Gameboy using TGB_Dual.

It additionally should support the following Mappers:

MBC2  Nintendo MBC 2
MBC4  Nintendo MBC 4

MBC7	Nintendo MBC7 (Tilt sensor cart)
